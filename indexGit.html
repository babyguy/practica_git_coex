Aprendiendo Git

Git es un sistema de control de versiones donde se puede almacenar proyectos grandes o pequeños, se usa para el desarrollo de proyectos e ir almacenando las actualizaciones de codigo de uno o varios desarrolladores, la creación, fusión y eliminación de esas líneas de codigo lleva unos segundos.
